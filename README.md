## Getting Started

### 1- **download the project** on your machine =)
```bash
git clone https://github.com/J0se-Luiz/user-details-page.git
```
<br/>

### 2- at the root of the project **yarn**
```bash
yarn
```

<br/>

### 3- Ready =) to run the local project, run **yarn start**

```bash
yarn start
```
